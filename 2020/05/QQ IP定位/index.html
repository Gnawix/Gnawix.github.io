通过QQ电话获取好友位置
发表于 2020-05-14| 更新于 2020-05-14
字数总计:180|阅读时长: 1 分钟
| 阅读量:

原理：通过与Wireshark抓取好友的IP地址，再通过IP地址实现定位

首先打开Wireshark，选择自己的连接方式

Ctrl+F选择过滤条件 显示过滤器选择字符串，然后选择分组详情，再输入020048

开始抓包

打开QQ，选择你的目标好友发起QQ通话。无论好友是否接听，只需成功发起通话即可。

此时返回Wireshark，不断点击查找。

拿到好友的ip后打开网站OpenGPS，输入好友的ip即可获得好友位置。

注意：此方法有误差，如果好友WIFI在线则误差较小，4G在线则会有较大误差，无法实现精准定位。

完
